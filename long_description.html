Landfill Analysis is a demo application for PyBossa that shows how you can crowdsourcing landfill data using Microsoft Bing images.

This application uses location information from the EPA regarding landfills across the United States. The information from the EPA specifies the latitude and longitude of landfills as well as other miscellaneous information about each individual landfill.

When the user adds a polygon or a point, the action saves the position and the coordinates in GeoJSON format to easily parse it at a later stage.
